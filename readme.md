# Gamebuino fonts

This repo contains the fonts of the Gamebuino and the tools to create new ones.

The included font are already in the Gamebuino library, no need to add them to your code. See [Gamebuino reference/gb.display.setFont](http://gamebuino.com/wiki/index.php?title=Gb.display.setFont)

The files without extensions are meant to be opened with GLCDFontCreator2.
Then you can export them to a .txt file and copy and paste them to your Gamebuino code.



Font editor from https://code.google.com/p/glcd-arduino/downloads/detail?name=GLCDFontCreator2.zip
Which is part of the [glcd-arduino](https://code.google.com/p/glcd-arduino/) library, licenced GNU Lesser GPL.
